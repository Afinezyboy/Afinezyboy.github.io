Latex学习笔记

2022-03-19
2 min read

环境

VSCode配置

  1. 下载latex-workshop

  2. 在“首选项-设置-扩展-Latex”中，输入以下json：

    "latex-workshop.latex.tools": [
	{
		"name": "xelatex",
		"command": "xelatex",
		"args": [
			"-synctex=1",
			"-interaction=nonstopmode",
			"-file-line-error",
			"%DOCFILE%"
		]
	},
	{
		"name": "pdflatex",
		"command": "pdflatex",
		"args": [
			"-synctex=1",
			"-interaction=nonstopmode",
			"-file-line-error",
			"%DOCFILE%"
		]
	},
	{
		"name": "bibtex",
		"command": "bibtex",
		"args": [
			"%DOCFILE%"
		]
	}
],

"latex-workshop.latex.recipes": [
	{
		"name": "xelatex",
		"tools": [
			"xelatex"
		],
	},
	{
		"name": "pdflatex",
		"tools": [
			"pdflatex"
		]
	},
	{
		"name": "xe->bib->xe->xe",
		"tools": [
			"xelatex",
			"bibtex",
			"xelatex",
			"xelatex"
		]
	},
	{
		"name": "pdf->bib->pdf->pdf",
		"tools": [
			"pdflatex",
			"bibtex",
			"pdflatex",
			"pdflatex"
		]
	}
],

命令

插入文件

input命令可以改为include，区别在于，input可以放在导言区和正文区，包含的内容不另起一页；而include只能放在正文区，包含的内容另起一页。

宏包

手动安装宏包

有些宏包无法直接安装，必须通过从CTAN上下载手动安装。以tools宏包为例，首先在CTAN上搜索到该宏包，进入下载后得到tools.zip文件。解压缩，用cmd命令进入解压缩后的文件夹，然后输入：

>> latex tools.ins

经过一段时间编译，tools.sty以及该宏包下其他子宏包就生成了。接着需要把整个文件夹复制粘贴至...\MiKTeX\tex\latex\文件夹中，注意还有最后的关键一步，打开MikTeX软件，在“宏包”选项卡中点击“刷新数据库”，更新数据库中的宏包，才能在MikTeX中看到手动安装的宏包。
