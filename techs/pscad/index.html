自定义模块编程

规范

已有变量

名称 符号
仿真时间 TIME
仿真步长 DELT

普通变量

定义局部变量：

#LOCAL REAL var1,var2,var3

定义输出变量OUT1，并将var1的值赋给它：

#OUTPUT REAL OUT1 {var1}

历史变量

历史数据需要使用#STORGAE定义，数据类型包括：

如果不需要对数据进行初始化，则可采用Inter Time Step Data Transfer中的数据类型。例如下面的模块定义了一个积分器，其增益由输入变量$IN控制：

#STORAGE REAL:1
! This means we use one storage for a real variable.
! The corrsponding memery name is 'STORF'
!     and NSTORF is the counter.
! remember DO NOT insert blankets into 'REAL' and '1'.
#LOCAL REAL LAST
        LAST = STORF(NSTORF)
        LAST = $IN*DELT + LAST
        $OUT = LAST
! Remember to add the correct counter to NSTORF 
!     in order that the memery works well
        STORF(NSTORF) = LAST
        NSTORF = NSTORF + 1

如果需要在$t=0$时刻进行初始化，则可以使用BEGIN to DSDYN/DSOUT Data Transfer中的数据类型，并将初始化部分放在#BEGIN...#ENDBEGIN结构中。例如下面定义了一个含初始值的积分器，其初值由参数$init定义：

! Use #BEGIN routine for initialization
#BEGIN
        RTCF(NRTCF) = $init
        NRTCF= NRTCF + 1
#ENDBEGIN

#STORAGE RTCF:1
#LOCAL REAL LAST
        LAST = RTCF(NRTCF)
        LAST = $IN*DELT + LAST
        $OUT = LASTLAST
! Remember to add the correct counter
!     in order that the memery works well
        RTCF(NRTCF) = LAST
        NRTCF = NRTCF + 1

变量引用

使用局部变量时，直接用变量名称var1即可；使用从模块的输入输出变量或者参数定义变量时，要加上引用符号$。例如引用输入变量$IN1，参数$par1

目前查到的资料中，只有!这一种方式：

语句

判断语句

Fortran采用IF...THEN判断结构，中间可采用ELSE选择不满足的情况。注意判断语句结尾需要加上ENDIF。示例如下：

IF ($IN1 .GE. $IN2) THEN
    $OUT1 = $IN1
ELSE
    $OUT1 = $IN2
ENDIF

另外，判断语句可以使用ELSEIF进行嵌套。

循环语句

对参数$dim定义的每个数组进行操作：

#LOCAL INTEGER CNT
DO CNT=1,$dim
    ...
ENDDO

安装

安装包地址： mhi-pscad · PyPI 

pip install mhi-pscad

语法

加载Pscad

加载Pscad句柄（无论Pscad软件是否已经打开）

#!/usr/bin/env python3
import mhi.pscad 
with mhi.pscad.application() as pscad:
	# user command
    pass

如果没有打开软件，可以使用launch代替application：

with mhi.pscad.launch() as pscad:
	# user command
    pass

如果对当前已经打开的Pscad软件进行操作，也可以使用connect代替application：

with mhi.pscad.connect() as pscad:
	# user command
    pass

打开操作

加载workspace、project

pscad.load("<file folder>")

Project操作

获得project操作句柄

windfarm = pscad.project("windfarm")
feeder1 = pscad.project("feeder1")
# ...

修改project settings：

feeder1.parameters(time_step="10.0")
feeder1.parameters(time_duration="6.0")
feeder1.parameters(sample_step="200.0")

寻找canvas上的module，并修改参数。注意，如果需要使用find命令，则模块必须含有Name参数。

canvas = feeder1.canvas("Main")
line1 = canvas.find("master:newpi", "Line1")
line1.parameters(len="800 [m]")

查看参数：

params = line1.parameters()
print(params['Name'])

设置位置position

component.get_location()
component.set_location()

查询端口位置

port = component.port(port_name)
port.x		# 横坐标
port.y		# 纵坐标
port.dim	# 端口维数
