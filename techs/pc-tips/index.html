电脑操作

Windows全装机

  1. U盘启动盘准备

  2. 去MSDN官网（ https://msdn.itellyou.cn/ ）下载系统镜像。

  3. 在“老毛桃”官网（ https://www.laomaotao.net/ ）下载winpe。推荐下载“在线版”，这样可以仅下载必要的组件，节省空间和时间。

  4. 准备一个存储空间至少为8G的U盘，对其中的内容进行备份，因为制作U盘启动盘会清空其中的内容。将U盘插入电脑中。

  5. 打开已经下载好的老毛桃，将插入的U盘制作为U盘启动盘。注意，“请选择”栏是插入的U盘的名称，“格式”应该选择NTFS，以装下约4G大小的win10系统的映像文件。确定无误后，点击“一键制作成U盘启动盘”。

  6. 当制作U盘启动盘完成后，U盘的名称会变为“老毛桃U盘”。将准备好的windows系统映像ISO文件复制到U盘中“\LMT”文件夹下。

  7. 此时，一个U盘启动盘已经制作完毕，并且含有windows系统的安装文件。

  8. 设置BIOS

科普：电脑有两种引导方式（Boot Mode），分别是Legacy和UEFI。其中Legacy出现的时间很早，是传统BIOS启动的方式（开机→BIOS初始化→BIOS自检→引导操作系统→进入系统。）；而UEFI是一种较新的启动方式（开机→UEFI初始化→引导操作系统→进入操作系统启动），出现的时间较晚，仅支持64位的操作系统，但是特点是：快。除了这两种引导方式，还存在两种硬盘分区表格式，即MBR（主引导记录：Master Boot Record）和GPT（全局唯一标识分区表：GUID Partition Table）。其中，如果选择了采用UEFI的方式进行引导，就必须设置硬盘分区格式为GPT。对于大多数支持固态硬盘的电脑来说，采用UEFI方式引导，并对系统盘（通常为你的固态硬盘）进行GPT的分区方法。接下来是具体步骤：（以下步骤均需要保持U盘启动盘插入电脑）

  1. 按下开机键后，会显示电脑主板的标志（比如联想的会显示Lenovo，戴尔会显示Dell），此时按下键盘上的某个键，进入BIOS的SETUP界面，对BISO进行设置。（我的电脑是按下侧面的Reset键直接进入BIOS，如果不清楚可以百度）

  2. 进入BISO的“BOOT”栏，看看你的电脑是否支持UEFI的启动方法。若支持，则设置为UEFI启动方法，否则不做任何处理。保存并退出，电脑会自动重启。

  3. 当电脑再次显示主板标志后，按下“F2”键，此时进入BOOT MANAGER，选择“USB Device”的启动选项，进入PE系统。

  4. 安装系统

  5. 考虑到我的电脑是120G固态+1T机械，所以用Disk genius 软件将固态硬盘的分区表改成GPT格式。选择移动硬盘，点击“快速分区”，注意选择“GUID”格式，这就表示采用了GPT分区。一般只需要创建一个分区就行了（其他内容不要改动，让Disk genius自动创建ESP和MSR两个小分区，用来引导UEFI的启动方式）。在这个过程中会抹去固态硬盘上的所有数据。

  6. 接下来，双击桌面上的“老毛桃一键装机”，原先的iso文件已经被老毛桃识别出来，选择将系统安装到C盘，点击“执行”。

  7. 等待老毛桃自动安装，一切完毕后重启电脑即可。

Windows 11 恢复完整右键菜单

Windows 11 的右键菜单总是显示不完整，比较反人类。通过在cmd中输入下方命令：

**reg add HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32 /ve /d “” /f**

再通过ctrl+shift+esc打开任务管理器，重启资源管理器(explorer.exe)即可。

如果想要回到Windows 11原来的样式，则在cmd中输入以下命令即可：

reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f

Word样式

合并多个Word文件

打开一个新的Word文档，依次点击菜单栏中的“插入-对象-文件中的文字”，选择需要合并的文件，就能把这些文件合并在一个Word文档中。

各个文件最好使用同一个Word模板中的样式。如果需要对合并后的文件套用模板，最好通过“样式集”的方式进行套用。方法如下：首先要有一个模板文件（或者已经调整好样式的Word文档），依次点击菜单栏中的“设计-文档格式-另存为新样式集”，将该文档中的样式保存至默认位置；接着，在需要套用新样式集的Word文档中，在菜单栏的“设计-文档样式”中选中刚才所保存的样式文件，就能够将本文档中的样式替换为该样式集中的样式，从而完成模板替换。

Office公式字体美化

Word排版的公式采用Axmath软件编辑。当定稿后，利用Axmath的宏将全文的公式转换为Office Math。在转换之前，设置默认字体为“Latin Modern Math”。该字体一般保存在我的收藏夹内。如果遗失，可前往 Latin Modern Math 下载该字体。

Axmath与Mathtype之间的转换

中文期刊接受哪种数学编辑软件？

选择Word而非Latex

在毕业之前，所有期刊投稿均采用Word排版而非Latex排版。不再耗费脑力去学习Latex使用方法，也不用担心导师修改问题。

输出PDF

使用Adobe PDFmaker

如果使用Office Math且包含默认字体之外的字体，那么必须使用Adobe PDFmaker进行PDF转换，才能保留矢量公式信息，否则会被转换为位图。

但是另一方面，使用Adobe PDFmaker会导致Matlab直接插入的图片中的字体变小。经过测试，将Matlab图片另存为SVG格式可以避免字体缩小问题，但网上说的转换为EMF格式不行。另外，如果Visio中的图片太大的画，不利于编辑；而如果转化为PNG，则在转换后容易变得模糊。为此建议：

  • MATLAB中的图片统一保存为SVG格式，粘贴至Word中；
  • Visio中的大图片统一保存为emf格式

参考文献超链接

在EndNote选项卡中，点击左侧“Edit & Manage Citations(s)”，在弹出框左下角“Tools-Configure Bibliography”，接着把最下方的“Link in-text citations to reference in the bibliography”勾上，这样就能够在输出的PDF中建立参考文献的超链接。如果继续勾选“Underline linked in-text citations”，那么参考文献的数字颜色将变成蓝色。

导入包的设置

  • 目录只有包含一个叫做 init.py 的文件才会被认作是一个包；
  • 如果包定义文件 init.py 存在一个叫做 all 的列表变量，那么在使用 from package import * 的时候就把这个列表中的所有名字作为包内容导入。
